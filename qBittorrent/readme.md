## You should apply the theme manually
1. Open qBittorrent and go to **Tools** > **Preferences** > **Behaviour**.
2. Tick the **"use custom UI theme"** checkbox and select your chosen `qBittorrent/themes/'Catpuccin Mocha v2.0.1..qbtheme` theme file.
3. Click **Apply** and restart qBittorrent for the changes to take effect.
